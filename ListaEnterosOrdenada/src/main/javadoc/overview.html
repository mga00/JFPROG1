Hello world! Proyecto para ser usado como base para un proyecto Java, puede ser usado como proyecto de tipo ejecutable o como proyecto de tipo librería/API

Características del proyecto Maven:

  • Maven, minimo requerido 3.6.3
  • Java 17
  • JUnit 5
  • Checkstyle se ejecuta como reporte Maven si encuentra un fichero de configuración checkstlyle en etc/daw_checks.xml
  • JaCoCo
  • Generación de Javadoc como JAR independiente
  • Generación de Javadoc como reporte Maven, con UMLdoclet
  • Reportes surefire/JUnit mejorados para usar @DisplayName tanto en consola como en XML
  • Clase main preconfigurada a com.gsd.daw.prog.App

Para usarlo tienes que cambiar lantolin por tu id1 en:

  • El nombre de proyecto Eclipse, si el proyecto aun no está en Git esto cambiará también el nombre del directorio base del proyecto.
  • El groupId del pom.xml

Para proyectos de tipo ejecutable, el proyecto se puede ejecutar de dos modos:

mvn exec:java -Dexec.mainClass=com.gsd.daw.prog.App -Dexec.args="arg1 arg2 ..."
